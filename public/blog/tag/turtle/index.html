Playing with pythons and turtles

30th April, 2014

For this week's CodeClub lesson we looked at the python library turtle. It's a easy and fun little graphics library based around moving a "turtle" cursor around the screen, drawing lines and filling in shapes.

For something that is so basic in functionality it's surprising how the results can be so satisfying.

You can check out the code at github.
